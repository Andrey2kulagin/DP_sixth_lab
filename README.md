Для того, чтобы скачать папку с файлами надо перейти по ссылке https://github.com/Andrey2kulagin/DP_sixth_lab и нажать на зелёную кнопку <>code, далее нажать Download ZIP и разархивировать скаченный архив. Ничего больше скачивать и устанавливать не требуется.
В скаченной папке есть файл sixth_lab.exe Если кликнуть по нему, то запускается программа она может работать с случайными данными, а может читать данные из файлов 6_x_data.txt 6_y_data.txt, по умолчанию в файлах стоят данные из Excel, в файлы, где стоят целые числа нужно вставлять только целые, в файлы, где дробные - только дробные. В качестве разделителя для дробный чисел надо использовать точку, в конце не должно быть пустой строки.
В папке лежит файл laboratornye_raboty_5_i_6_2.xlsx в нем можно сравнить работу программы и файла.
Если в лабораторной работе предполагается несколько графиков, то для перехода к следующему надо закрыть окно с текущим.
Код программы лежит в файле sixth_lab.py
